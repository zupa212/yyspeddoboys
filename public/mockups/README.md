# Mockups Directory

Place your phone mockup screenshots here. The hero section expects:

- `phone-1.png` — Left/main phone screen (e.g., map view, order tracking, dashboard)
- `phone-2.png` — Right/overlapping phone screen (e.g., restaurant detail, order status)

**Recommended specs:**
- Resolution: 1290 × 2796px (iPhone 15 Pro Max)
- Format: PNG with transparency, or WebP
- Content: Dark-mode app screens for best visual impact
